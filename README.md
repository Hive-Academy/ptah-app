## Download Desktop App

Ptah Desktop is a standalone desktop application — no VS Code required. Available for Windows, macOS, and Linux.

### Latest Release

Download the latest version from the [Releases page](https://github.com/Hive-Academy/ptah-app/releases/latest).

| Platform | File | Notes |
| -------- | ---- | ----- |
| **Windows** | `Ptah-Setup-x.x.x.exe` | NSIS installer |
| **macOS** | `Ptah-x.x.x.dmg` | Disk image (Intel & Apple Silicon) |
| **Linux** | `Ptah-x.x.x.AppImage` | AppImage (universal) |
| **Linux** | `ptah-desktop_x.x.x_amd64.deb` | Debian/Ubuntu package |

> **Note:** Builds are currently unsigned. You may see platform warnings ("unidentified developer") on first launch. Code signing will be added in a future release.

### Installation

**Windows**: Download and run the `.exe` installer. Choose your installation directory.

**macOS**: Download the `.dmg`, open it, and drag Ptah to your Applications folder. On first launch, right-click → Open to bypass Gatekeeper.

**Linux (AppImage)**: Download, make executable (`chmod +x Ptah-*.AppImage`), and run. Or use the `.deb` package: `sudo dpkg -i ptah-desktop_*.deb`.

## What is Ptah?

Ptah is an AI coding assistant that orchestrates multiple specialized agents to help you write, refactor, debug, and understand code. Key features:

- **14 Specialized AI Agents** — each trained for specific tasks (architecture, testing, security, etc.)
- **Workspace Intelligence** — automatic project detection and context-aware assistance
- **Built-in MCP Server** — code execution, workspace search, diagnostics, and more
- **Multi-Provider Support** — Claude, GPT, Gemini, and OpenRouter
- **14-Day Free Trial** — full Pro features, no credit card required

Learn more at **[ptah.live](https://ptah.live)**
